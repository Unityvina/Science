# Computer Science - Khoa học Máy tính

**English** | **Tiếng Việt**

---

## Introduction - Giới thiệu

**English**:  
Computer science is the study of computation, information processing, and the design of systems that automate tasks through algorithms and data structures. It bridges theory, hardware, and software to solve problems efficiently.

**Tiếng Việt**:  
Khoa học máy tính là ngành nghiên cứu về tính toán, xử lý thông tin và thiết kế các hệ thống tự động hóa công việc thông qua thuật toán và cấu trúc dữ liệu. Nó kết nối lý thuyết, phần cứng và phần mềm để giải quyết vấn đề một cách hiệu quả.

---

## Core Subfields - Các lĩnh vực cốt lõi

| **English** | **Tiếng Việt** |
|-------------|----------------|
| **Algorithms and Data Structures**: Foundations for efficient problem-solving (e.g., sorting with \(O(n \log n)\) time via merge sort; structures like hash tables for \(O(1)\) average lookups). | **Thuật toán và Cấu trúc dữ liệu**: Nền tảng giải quyết vấn đề hiệu quả (ví dụ: sắp xếp với độ phức tạp \(O(n \log n)\) bằng merge sort; cấu trúc như bảng băm cho tra cứu trung bình \(O(1)\)). |
| **Programming Languages**: From low-level (C/assembly) to high-level (Python/JavaScript); concepts like recursion, OOP, and functional programming. | **Ngôn ngữ lập trình**: Từ cấp thấp (C/assembly) đến cấp cao (Python/JavaScript); các khái niệm như đệ quy, lập trình hướng đối tượng (OOP), lập trình hàm. |
| **Computer Architecture**: CPU design, memory hierarchy (cache, RAM), parallelism (multi-core, GPUs). | **Kiến trúc máy tính**: Thiết kế CPU, hệ thống bộ nhớ phân cấp (cache, RAM), xử lý song song (đa lõi, GPU). |
| **Operating Systems**: Process management, scheduling, file systems (e.g., Linux kernel). | **Hệ điều hành**: Quản lý tiến trình, lập lịch, hệ thống tệp (ví dụ: kernel Linux). |
| **Networks and Security**: TCP/IP, encryption (RSA, AES), cybersecurity threats like SQL injection. | **Mạng và An ninh**: TCP/IP, mã hóa (RSA, AES), các mối đe dọa an ninh như tấn công SQL injection. |
| **Databases**: SQL/NoSQL, normalization, ACID properties. | **Cơ sở dữ liệu**: SQL/NoSQL, chuẩn hóa, tính chất ACID. |
| **Artificial Intelligence/Machine Learning**: Neural networks, reinforcement learning (e.g., transformers in LLMs). | **Trí tuệ nhân tạo/Học máy**: Mạng nơ-ron, học tăng cường (ví dụ: mô hình transformer trong các mô hình ngôn ngữ lớn). |
| **Theory of Computation**: Automata, Turing machines, P vs. NP problem. | **Lý thuyết tính toán**: Ôtômát, máy Turing, bài toán P vs. NP. |
| **Software Engineering**: Agile, version control (Git), testing. | **Kỹ thuật phần mềm**: Agile, quản lý phiên bản (Git), kiểm thử. |

## Key Milestones - Các mốc son quan trọng

| Year | **English** | **Tiếng Việt** |
|------|-------------|----------------|
| 1936 | Alan Turing's **Turing Machine** formalizes computation. | Alan Turing công bố **Máy Turing**, chính thức hóa khái niệm tính toán. |
| 1940s | **ENIAC**, the first electronic computer. | **ENIAC**, máy tính điện tử đầu tiên. |
| 1969 | **ARPANET** (precursor to the Internet). | **ARPANET** (tiền thân của Internet). |
| 1970s | **C language**, **Unix OS**. | Ngôn ngữ **C**, hệ điều hành **Unix**. |
| 2000s | Cloud computing, big data (**Hadoop**). | Điện toán đám mây, dữ liệu lớn (**Hadoop**). |
| 2010s–now | AI boom (deep learning), quantum computing (e.g., **Google Sycamore** quantum supremacy in 2019). | Bùng nổ AI (học sâu), tính toán lượng tử (ví dụ: **Google Sycamore** đạt ưu thế lượng tử năm 2019). |
